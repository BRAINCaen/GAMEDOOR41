L'EVJF dont tout le monde parlera

Vous organisez un enterrement de vie de jeune fille à Caen et vous cherchez une activité originale ? Chez GAMEDOOR•41, on propose deux formats parfaits pour les EVJF : un quiz game univers mariage (blind test mariage, défis, place mariée offerte avec le code MARIAGE) et un escape game pour jouer en équipes.

Le quiz est le format star des EVJF : le chauffeur de salle lance l'émission ; ensuite blind test mariage, défis sur le mariage et jokers s'enchaînent sur écran avec les buzzers. Fous rires garantis. L'escape game ajoute une dose d'adrénaline et de coopération. Vous pouvez combiner les deux pour une après-midi complète.

Questions fréquentes — EVJF

Combien de personnes pour un EVJF ?

De jusqu'à 46 participantes (escape + quiz). Le quiz game est idéal pour les grands groupes. L'escape game accueille 2 à 6 joueuses par salle (jusqu'à 30 sur 5 salles en simultané, 3 univers thématiques).

C'est quoi le quiz EVJF ?

Un univers thématique mariage prêt à jouer : blind test mariage (chansons d'amour, génériques de mariage), défis sur le mariage, ambiance plateau TV avec buzzers et chauffeur de salle. Place offerte pour la mariée avec le code MARIAGE sur présentation d'un accessoire « reine du jour » à l'arrivée (voile, couronne, écharpe… pas besoin de déguisement complet) — le chauffeur de salle s'occupe du reste.

Peut-on combiner escape + quiz ?

C'est le format le plus demandé pour les EVJF : escape game en petits groupes puis quiz toutes ensemble. Contactez-nous pour un devis combiné.

Quel est le prix ?

Quiz game : à partir de 18€/pers. Escape game : à partir de 24€/pers. Forfait combiné sur devis. Tarifs dégressifs pour les grands groupes.

L'EVJF est-il accessible si une participante est sourde ou malentendante ?

Le quiz Buzz Your Brain repose sur des contenus essentiellement audio (questions à l'oral, blind test) et n'est pas adapté aux personnes sourdes ou malentendantes. En revanche, nos escape games reposent sur l'observation et la coopération visuelle, mieux adaptés. Contactez-nous pour adapter votre EVJF.
