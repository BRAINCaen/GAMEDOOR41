# Posts Google Business Profile — Prêts à coller

Dossier de Posts GBP pré-rédigés pour la fiche **GAMEDOOR·41** sur Google Business Profile (business.google.com).

## Comment publier — 2 options

### Option A — Manuel (3 minutes par post)

1. Va sur https://business.google.com et sélectionne la fiche **GAMEDOOR·41**
2. Onglet **Publications** ou **Mises à jour** → **Ajouter une nouveauté**
3. Choisis le type : **Nouveauté** (ou "Offre" / "Événement" selon le cas, indiqué dans chaque fichier)
4. Ouvre le fichier `.md` correspondant → copie-colle :
   - Le **TITRE** dans le champ titre
   - Le **CORPS** dans le champ description
   - Sélectionne le **CTA** indiqué (Réserver / En savoir plus / etc.) et colle le **LIEN**
   - Upload l'**IMAGE** depuis le chemin local indiqué (`C:\Users\Direction Brain Caen\Desktop\gamedoor41\img\...`)
5. **Publier**

### Option B — Claude Chrome (pour publier plusieurs posts d'un coup)

Prompt-type à passer à Claude Chrome :

```
Aide-moi à publier N posts sur Google Business Profile (fiche GAMEDOOR·41).

```

## Format Posts GBP — rappels Google

- **Titre** : 58 caractères max recommandé (sinon coupé en miniature mobile)
- **Corps** : 1500 caractères max
- **Image** : ratio 1:1 ou 16:9, min 250×250 px, recommandé 1200×900 px (JPG/PNG)
- **CTA disponibles** : Réserver · En savoir plus · Acheter · S'inscrire · Appeler · Commander
- **Durée mise en avant** : 7 jours (puis le post reste consultable mais n'est plus en avant)
- **Pas de hashtags** ni de boucles d'emoji excessives (Google dévalorise)

## Tracking

Chaque lien dans les posts utilise `?utm_source=gbp&utm_medium=post&utm_campaign=NOM_DU_POST` pour pouvoir suivre les conversions GBP → site dans Google Analytics 4 (G-2ENESBTXTE).
