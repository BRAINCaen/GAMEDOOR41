## 🥇 PRIORITÉ 1 — Tourisme local (DA très fort)

### 1. Caen-Tourisme.fr (Office de Tourisme Caen-la-Mer)

| Critère | Valeur |
|---|---|
| **DA estimée** | ~55 (officiel) |
| **URL inscription** | https://caen-tourisme.fr/professionnel-du-tourisme/ |
| **Type** | Référencement professionnel touristique officiel |
| **Coût** | Gratuit pour membres → adhérer ~150€/an (membre Office Tourisme) |
| **Difficulté** | Moyenne (validation manuelle 7-14j) |
| **Bonus** | Backlink officiel local + relais sur réseaux sociaux + brochures papier |

**Procédure** : adhésion via formulaire ou contact direct `pro@caen-tourisme.fr`. Demander la **fiche pro escape game / loisirs immersifs**. Très valorisé par Google (autorité locale officielle).

**À fournir** : SIRET, attestation TVA, photos haute résolution, description bilingue FR/EN si possible.
