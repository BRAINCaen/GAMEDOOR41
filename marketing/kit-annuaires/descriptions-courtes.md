## 💡 Astuce — choisir la bonne longueur

| Annuaire type | Longueur conseillée |
|---|---|
| Annuaire automatique / formulaire rapide | 100-200 char |
| Fiche standard (Yelp, PagesJaunes) | 200-400 char |
| Tripadvisor / GBP description | 500-750 char |
| Petit Futé / Le Routard / éditorial | 1000-2500 char |
| Office du Tourisme | 750-1500 char |

**Règle générale** : toujours mettre les **mots-clés principaux dans les 100 premiers caractères** (escape game, Caen, Mondeville). Google indexe en priorité les premiers mots de chaque description.
