Questions fréquentes — EVG

Combien de personnes pour un EVG ?

De jusqu'à 46 participants (escape + quiz). Escape game : 2 à 6 par salle (30 max sur 5 salles, 3 univers thématiques). Quiz game : 4 à 16 candidats par session (8 pupitres, binôme possible).

C'est quoi le quiz EVG ?

Un univers thématique mariage prêt à jouer : blind test mariage, défis sur le mariage, ambiance plateau TV avec buzzers et chauffeur de salle. Place offerte pour le marié avec le code MARIAGE sur présentation d'un accessoire « roi du jour » à l'arrivée (nœud papillon, couronne, écharpe… pas besoin de déguisement complet) — le chauffeur de salle s'occupe de mettre l'ambiance.

Quel est le prix ?

Quiz game : à partir de 18€/pers. Escape game : à partir de 24€/pers. Forfait combiné sur devis. Tarifs dégressifs pour les grands groupes.

L'EVG est-il accessible si un participant est sourd ou malentendant ?

Le quiz Buzz Your Brain repose sur des contenus essentiellement audio (questions à l'oral, blind test) et n'est pas adapté aux personnes sourdes ou malentendantes. Nos escape games reposent sur l'observation et la coopération visuelle, mieux adaptés. Contactez-nous pour adapter votre EVG.
