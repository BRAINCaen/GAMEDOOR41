L'EVG qui met tout le monde d'accord

Vous cherchez une activité d'enterrement de vie de garçon à Caen qui claque ? Chez GAMEDOOR•41, deux options : un escape game intense avec game master acteur (le marié enfermé dans une cellule, ça fait toujours effet) ou un quiz game compétitif avec buzzers, blind test et défis personnalisés sur le futur marié.

Le combo escape + quiz est notre format le plus demandé pour les EVG : adrénaline d'abord, puis fous rires et compétition. De 2 à 40 participants.

Questions fréquentes — EVG

Combien de personnes pour un EVG ?

De 2 à 40 participants. Escape game : 2 à 6 par salle (18 max sur 3 salles). Quiz game : 4 à 40 joueurs en une session.

Le quiz est-il personnalisé ?

Oui ! Envoyez-nous les anecdotes et infos sur le futur marié. On intègre tout dans le quiz : questions personnalisées, blind test, défis.

Quel est le prix ?

Quiz game : à partir de 18€/pers. Escape game : à partir de 25€/pers. Forfait combiné sur devis. Tarifs dégressifs pour les grands groupes.
