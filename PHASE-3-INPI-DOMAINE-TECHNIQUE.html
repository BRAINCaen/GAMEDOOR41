GAMESIDE vs GAMEDOOR — Risque de confusion ?

GAMESIDE est déposé en classe 41 (loisirs/escape game) par une entreprise de Perpignan. Les noms sont proches phonétiquement mais suffisamment distincts (DOOR ≠ SIDE). Le risque d'opposition est faible mais pas nul. Un dépôt rapide et en bonne et due forme vous protège.

Procédure d'achat — OVH (recommandé)

1
Commander le domaine
  1. Aller sur www.ovhcloud.com/fr/domains/
  2. Taper gamedoor41 dans la recherche
  3. Sélectionner gamedoor41.fr + gamedoor41.com
  4. NE PAS prendre d'hébergement (vous êtes sur Netlify)
  5. Prendre l'option DNS anycast si proposée (+2€/an, ça accélère)
  6. Renseigner les infos du titulaire : SARL BOEHME
  7. Payer (~20€ total pour les 2 domaines la première année)
2
Configurer les DNS pour Netlify

Dans le panneau OVH > Domaines > gamedoor41.fr > Zone DNS, ajouter :

# Enregistrements DNS à ajouter chez OVH # Option A : Utiliser Netlify DNS (recommandé) # → Dans Netlify : Settings > Domain management > Add custom domain # → Netlify vous donnera des nameservers à configurer chez OVH # → Chez OVH : changer les serveurs DNS vers ceux de Netlify dns1.p01.nsone.net dns2.p01.nsone.net dns3.p01.nsone.net dns4.p01.nsone.net # Option B : Garder les DNS OVH et pointer vers Netlify Type Nom Valeur A @ 75.2.60.5 CNAME www gamedoor41.netlify.app. # L'option A est plus simple et gère automatiquement le HTTPS
3
Configurer Netlify
  1. Dans Netlify > Site settings > Domain management
  2. Cliquer "Add custom domain"
  3. Entrer : gamedoor41.fr
  4. Ajouter aussi : www.gamedoor41.fr (redirect auto vers gamedoor41.fr)
  5. Netlify génère automatiquement le certificat SSL (HTTPS gratuit via Let's Encrypt)
  6. Attendre la propagation DNS : 15 min à 48h (généralement 1-2h)
4
Configurer la redirection gamedoor41.com → gamedoor41.fr

Le .com doit rediriger automatiquement vers le .fr (un seul site, pas de duplicate content).

  1. Ajouter gamedoor41.com comme domaine alias dans Netlify
  2. Netlify redirigera automatiquement vers le domaine principal (.fr)

Partie C : Bascule technique

Couper les anciens sites Wix et activer les redirections 301

RÈGLE ABSOLUE : Ne jamais basculer un vendredi ou un week-end

Si quelque chose casse, vous devez pouvoir réagir. Mardi ou mercredi matin = idéal.

Pré-requis avant la bascule

  • Domaine gamedoor41.fr acheté et fonctionnel (HTTPS actif)
  • Fiche Google renommée en GAMEDOOR·41 et validée
  • Nouveau site testé à fond sur gamedoor41.fr
  • Toutes les canonical URLs mises à jour vers gamedoor41.fr
  • Sitemap.xml mis à jour avec les URLs gamedoor41.fr
  • Google Search Console configurée pour gamedoor41.fr
