Vous préparez un mariage à Caen et vous cherchez une activité qui fédère vraiment vos invités ? Pas une animation qui dure 10 minutes pour finir oubliée à la fin du repas.

Chez GAMEDOOR·41, on accueille les mariés et leurs invités sur tous les moments forts du mariage : l'EVJF entre copines, l'EVG entre potes, le jour J pendant que le photographe vole les mariés, le brunch du lendemain pour décompresser. FAQ Mariage

Questions fréquentes — Mariage à Caen

Quels événements liés au mariage organisez-vous ?

Tout ce qui tourne autour du mariage : EVJF, EVG, jour J (animation des invités après le vin d'honneur ou avant le repas), brunch du lendemain, pré-mariage entre proches. Quiz univers mariage avec blind test mariage et défis sur le mariage, escape game en équipes pour vos invités. Jusqu'à 46 personnes en simultané (30 escape + 16 quiz).

La mariée et le marié bénéficient-ils d'un avantage ?

Oui. Avec le code MARIAGE, la place du futur·e marié·e est offerte sur le quiz et sur l'escape game. Condition : arriver avec un accessoire « reine du jour » ou « roi du jour » (voile, nœud papillon, couronne, écharpe… au choix, on n'impose pas le déguisement complet). Valable en EVJF, EVG ou jour J.

Combien de personnes peuvent participer ?

Jusqu'à 46 personnes en simultané : 30 max sur les 5 salles d'escape (3 univers thématiques, 2 à 6 par salle) + 16 max sur le plateau Quiz Buzz Your Brain. Au-delà, on cale plusieurs créneaux dans la journée pour accueillir tous les invités.

Quels sont les tarifs pour un mariage ?

Quiz univers mariage : 15€ TTC/pers. Escape game : 25€ TTC/pers. (grille publique particuliers). Combo escape + quiz pour vos invités : sur devis. Place mariée·marié offerte avec le code MARIAGE (sur présentation d'un accessoire reine/roi du jour). Privatisation possible pour la journée.

Le quiz est-il sur-mesure pour les mariés ?

Le Quiz EVJF/EVG est un univers thématique mariage prêt à jouer (blind test mariage, défis sur le mariage, ambiance plateau TV) — pas un quiz personnalisé sur le couple. La place offerte avec le code MARIAGE et l'univers thème mariage suffisent à créer le moment fort.

Peut-on organiser un brunch ou un repas lendemain de mariage ?

Oui — le brunch du lendemain ou la journée détente entre proches est un format apprécié. Privatisation de l'espace possible avec restauration (cocktails, plateaux, viennoiseries, boissons) pour 20 personnes minimum sur cocktail traiteur. Devis sur-mesure.

Mariage à Caen : EVJF, EVG, jour J et lendemain chez GAMEDOOR·41

GAMEDOOR·41 est le lieu de référence pour les animations mariage à Caen et en Normandie. Au 41 bis rue Pasteur à Mondeville (5 min de Caen centre · grand parking · bus ligne 9 · sortie périphérique Mondeville), notre espace propose deux formats parfaits pour les mariages : le Quiz Buzz Your Brain univers mariage (plateau TV, blind test mariage, défis sur le mariage diffusés sur écran après le lancement par le chauffeur de salle) et l'escape game Brain Escape Game (3 univers thématiques sur 5 salles physiques avec game master acteur, jusqu'à 30 personnes en simultané).

Le combo escape + quiz permet d'accueillir jusqu'à 46 invités en simultané sur la même demi-journée — format idéal entre la cérémonie et le vin d'honneur, ou pour le brunch du lendemain. Avec le code MARIAGE, la place du futur·e marié·e est offerte (sur présentation d'un accessoire reine/roi du jour). Tous les moments du mariage sont couverts : EVJF, EVG, jour J, brunch du lendemain, pré-mariage entre témoins, vin d'honneur.

Pas de tarif fixe mariage : c'est la grille tarifaire particulier (dégressive selon la taille du groupe) qui s'applique, + place mariée·marié offerte avec le code MARIAGE et toutes les réductions cartes habituelles (CEZAM, Trip Normand, Pass Ambassadeur Normandie, City Pass, Réduc de Normandie, CSE…). Privatisation totale possible pour le jour J ou le lendemain (avec restauration : cocktails, plateaux, viennoiseries). Contactez-nous ou demandez un devis — réponse sous 48h ouvrées.
