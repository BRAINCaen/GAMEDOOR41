Questions fréquentes — Anniversaire

À partir de quel âge ?

Quiz game : dès 7 ans. Escape game Back to the 80's : dès 10 ans (accompagné d'un adulte). Garde à Vue : dès 14 ans. Psychiatric : dès 12 ans.

Combien d'enfants peuvent participer ?

Quiz game : de 4 à 40 enfants. Escape game : 2 à 6 par salle (jusqu'à 18 sur 3 salles). Le quiz est idéal pour les grands groupes d'anniversaire.

Le quiz est-il adapté aux enfants ?

Oui ! Le Quiz Kids est spécialement conçu : questions adaptées, ambiance fun, buzzers. Les enfants adorent. Le présentateur gère tout du début à la fin.

Quel est le tarif ?

Quiz game : à partir de 18€/pers. Escape game : à partir de 25€/pers. Tarifs dégressifs pour les grands groupes. Contactez-nous pour un devis anniversaire.

L'anniversaire est-il accessible si un enfant est sourd ou malentendant ?

Le Quiz Kids repose sur des contenus essentiellement audio (questions à l'oral, blind test) et n'est pas adapté aux enfants sourds ou malentendants. L'escape game Back to the 80's, basé sur l'observation et la manipulation d'objets, est mieux adapté. Contactez-nous pour adapter l'anniversaire.
